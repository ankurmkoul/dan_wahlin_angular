Filters

Filters allow us to change data using | syntax - you can define your own ones but here I'm going to show some built in ones

raw value1: {{value1}}

value1 as uppercase: {{value1|uppercase}}

raw value2 : {{value2}}

value2 as currency : {{value2|currency}}

value2 as currency with £ option passed as a parameter: {{value2|currency:"£"}}

note that the code uses ng-init to set the data - this is pretty bad practise but its a handy way to chuck data into a scope with creating controllers etc
