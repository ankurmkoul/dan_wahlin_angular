Filters allow us to change data using | syntax - you can define your own ones but here I'm going to show a couple (there aren't many) built in ones

raw value1: {{value1}}

value1 as uppercase: {{value1|uppercase}}

raw value2 : {{value2}}

value2 as currency : {{value2|currency}}

value2 as currency with £ option passed as a parameter (using:): {{value2|currency:"£"}}

value2 as currency with some random string option passed as a parameter (using:): {{value2|currency:"some random string"}}

a date formatted: {{'2015-12-12' | date:'yyyy'}}

note that this date has to be in this format: yyyy-mm-dd
