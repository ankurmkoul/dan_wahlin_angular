Basic functionality

We only need to include and import the angular script to get access to lots of basic angular goodies:

  • directives:
    • ng-model
    • ng-init
    • ng-non-bindable
    • ng-show - this shows if the variable is not an empty string
    • ng-hide - this hides when the variable is an empty string
  • data binding (and {{this syntax}})
