Basic functionality

this example uses a couple of components to show user interactions updating a simple boolean field

the checkbox is bound the "myValue" using the ng-model directive

the button uses ng-click to run myValue!=myValue to reverse the value of the boolean

my value = {{myValue}}

an alternative way to output a variable is using ng-bind - its bound to the following span - my value =

this one shows how we can use ng-class to update how our ui looks based on what the user does

note that setting the class requires class="red" whereas ng-class requires ng-class="'red'" (or in this ase ng-class="myColour" which references 'red')

my colour: {{myColour}}
